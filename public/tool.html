Help & Instructions

How to use ClusterSEO Pro

  1. Upload a CSV/TXT file with keywords and optional volumes
  2. Or manually input keywords in the Manual Input tab
  3. Adjust clustering sensitivity to control grouping tightness
  4. Click "Start Clustering Analysis" to process
  5. View and export your intent-classified keyword clusters

Intent Classification

  • Informational: how, what, why, guide, tutorial
  • Transactional: buy, price, order, shop, cost
  • Commercial: best, review, compare, vs, top
  • Navigational: login, website, official, app

CSV Format

Header row: keyword,volume — then one keyword per row. Volume is optional.
